Селекторы потомков (descendent selectors)


li > span {
 /* Объявления стиля */
}
Дочерние селекторы (child selectors). Cправа налево, селектор выбирает все span, являющиеся непосредственными потомками элемента li.


header ~ ul {
 /* Объявления стиля */
}
Родственные селекторы (sibling selectors). Cелектор выберет все элементы ul, которым предшествует элемент header с тем же родительским элементом. 


li + li {
 /* Объявления стиля */
}
Смежные родственные селекторы (adjacent sibling selectors). Выбраны все элементы li, которым непосредственно предшествует имеющий того же родителя элемент li.
